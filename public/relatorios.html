Segurança de Firmware

67% dos dispositivos ESP32-CAM apresentam firmware desatualizado. Recomendamos atualização imediata para a versão 2.0.1 para corrigir vulnerabilidades críticas.

Configuração de Rede

3 dispositivos Raspberry Pi estão com portas SSH abertas para a internet. Recomendamos implementar VPN ou restringir acesso por IP.

Proteção de Dados

Sensores DHT22 estão transmitindo dados sem criptografia. Sugerimos implementar TLS para proteger dados sensoriais em trânsito.
