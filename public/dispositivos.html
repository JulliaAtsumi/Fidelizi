Dispositivos Suportados

ESP32-CAM

Microcontrolador com câmera — usado em testes de transmissão, autenticação e armazenamento de vídeo.

  • Conectividade: Wi-Fi
  • Protocolos: HTTP, MQTT
  • Riscos comuns: credenciais fracas, firmware não assinado

Raspberry Pi Zero W

Plataforma Linux leve para serviços locais, gateways e protótipos — bom para testes de integração e containers.

  • Conectividade: Wi-Fi, USB
  • Protocolos: SSH, HTTP, MQTT
  • Riscos comuns: portas abertas, credenciais padrão

Sensor DHT22

Sensor de temperatura/umidade. Ótimo para demonstrações de telemetria, autenticação e segurança dos dados sensoriais.

  • Conectividade: Serial / GPIO
  • Riscos comuns: falta de criptografia em links, ausência de autenticação
